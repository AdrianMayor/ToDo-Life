Lista de tareas

Aplicación web que permita llevar registro de tareas

pendientes.


Descripción

● La aplicación debe mostrar un formulario con una caja de texto
donde escribir el texto de la tarea (p.ej. Comprar pan). Debe limitar
el tamaño del texto. Un selector de prioridad de tarea (Normal o
Importante) y un botón para añadir la tarea.

● Debajo de la caja de texto debe mostrar la lista de tareas añadidas
con la fecha que se añadieron y la prioridad.

● Al lado de cada tarea debe aparecer un checkbox para marcar la
tarea como hecha (o desmarcarla)

● Las tareas marcadas como hechas deben aparecer visualmente
tachadas.

● La aplicación debe tener un botón que permita limpiar las tareas
marcadas como hechas (borrarlas de la lista)

● La lista de tareas debe guardarse en el almacenamiento del
navegador para que se mantenga al volver a abrir la página.
